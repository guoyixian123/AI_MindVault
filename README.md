介绍:本项目是基于langchain4j开发的个人法律智能体,采用流式调用,支持会话隔离,会话记忆持久化,数据向量化,rag知识库
它可以帮你解决一些生活上的问题如: 
🔹【劳动权益】  
→ 签订/解除劳动合同合法性判断（含试用期、竞业限制、经济补偿金计算）  
→ 工资拖欠、加班费主张、社保公积金投诉路径与证据清单  
→ 被违法辞退时的“N+1”“2N”适用条件及仲裁时效提醒（1年）  

🔹【消费维权】  
→ 网购七日无理由退货例外情形（定作/鲜活易腐/拆封影响二次销售等）  
→ 餐饮/医美/教培预付卡退费依据（《消费者权益保护法》第53条+地方条例）  
→ 直播带货虚假宣传的举证要点（录屏+订单+广告截图+聊天记录）  

🔹【民间借贷】  
→ 借条/欠条效力核心要素（金额大写、借款人亲签、无利息约定视为无息）  
→ 逾期利息计算标准（LPR四倍上限，2024年现行14.8%）  
→ 微信转账无备注时的补强证据建议（催款录音+书面确认）  

🔹【婚恋家事】  
→ 彩礼返还法定情形（未登记、登记未共同生活、婚前给付致生活困难）  
→ 婚内借款协议有效性（需明确为个人债务+资金用途证明）  
→ 抚养权争取关键证据（孩子实际抚养记录、稳定收入证明、教育医疗支出凭证）  

当然,本项目也可以自行配置,它可以成为任何角色,如个人医生,个人金融顾问,个人生活顾问等,

首先我们需要配置systeam角色设定
![示例1](%E6%88%AA%E5%B1%8F2026-03-27%2022.36.17.png)
然后我们要配置自己的rag知识库,防止ai幻觉,在content路径下放置数据文件
![示例2](%E6%88%AA%E5%B1%8F2026-03-27%2022.40.30.png)


#### 使用说明

1.  使用之前我们需要通过配置yml文件,添加自己的apikey,和自己的模型url和名称
![输入图片说明](%E6%88%AA%E5%B1%8F2026-03-27%2022.59.31.png)
2. 我们需要下载redis-stack(支持redis基础功能,是redis的升级)
3. 我们将content/文件下的数据进行向量化,确保存储到redis中(注意:将数据向量化并且保存到向量数据库之后,下次启动就可以把@bean注释掉,避免资源浪费——token!!!)![示例3](%E6%88%AA%E5%B1%8F2026-03-27%2022.41.42.png)这里的文件需要自行获取:https://modelscope.cn/datasets
4.恭喜你获得了自己的智能法律助手
